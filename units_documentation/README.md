## About Units

Units is a comprehensive warehouse management system that provides:

- **Real-time inventory tracking** across multiple warehouses
- **Inbound logistics management** from receiving to putaway
- **Outbound order fulfillment** with picking, packing, and dispatch
- **SKU and product catalog management**
- **Customer and supplier management**
- **Reporting and analytics**
- **Multi-language support** (English and Arabic)
- **Role-based access control**
